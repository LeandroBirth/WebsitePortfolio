ForumHUB - API REST

Completed during the ONE (Oracle Next Education) program, hosted on the Alura platform.

Overview

This project is a REST API for managing a forum where users can:

  • Create, view, update, and delete topics.
  • Add and manage responses to topics.
  • Authenticate and authorize users with JWT.

The API is built using Spring Framework and adheres to REST best practices with proper validations, database integration, and secure access control.

Main Endpoints

Method URL Description
[GET] /topics Lists all topics.
[GET] /topics/{id} Retrieves a specific topic.
[POST] /topics Creates a new topic.
[PUT] /topics/{id} Updates an existing topic.
[DELETE] /topics/{id} Deletes a topic.
[GET] topics/{id}/answers Lists all responses to a topic.
[POST] /topics/{id}/answers Adds a response to a topic.
[POST] /answers/{id}/answer_solution Marks a response as the topic solution.
[POST] /api/login Logs in to receive a JWT token.
[POST] /api/register Registers a new user.

Technologies Used

  • Language: Java
  • IDE: IntelliJ IDEA
  • Framework: SpringBoot
  • Spring Data JPA, Spring Boot Starter Web, MySQL, Flyway, Lombok, Spring Validation, Spring Boot Security, Java-JWT.

Currency Converter Application for Desktop

Description: A simple yet effective currency converter application built using JavaFX. This application allows users to seamlessly convert currencies with a clean and intuitive interface. It utilizes an updated API to fetch the most current currency exchange rates of the day, parsing the API's JSON response to perform accurate and reliable conversions.

Getting Started:

  • Prerequisites:
    • Java 21 or higher installed on your system.
    • Ensure that you have permission to execute shell scripts (Linux/Mac).
  • Setup:
    • Download and extract the CurrencyConverter.zip or CurrencyConverter.tar.gz file.
    • Open a terminal and navigate to the extracted directory.
  • How to Run:
    • Ensure execution permissions are set for run.sh using:
      • chmod +x run.sh
    • Run the following command to execute the application:
      • ./run.sh
    • Alternatively, use:
      • sh run.sh
