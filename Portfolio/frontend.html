Task Management Application

Description: Task Management is a full-stack web application designed to help users efficiently manage their tasks. The system allows users to create, update, and delete tasks, providing an intuitive interface and a robust back-end for seamless task organization.

Features:

  • User authentication (still under refinement)
  • Create, update, and delete tasks
  • Categorize tasks based on status or priority
  • Responsive UI for an optimized user experience
  • RESTful API for seamless communication between front-end and back-end
  • Database persistence for storing tasks

Technologies Used:

  • Front-End: React (TypeScript), React Router, Node.js, CSS
  • Back-End: Java Spring Boot, Spring Security (authentication still being refined), PostgreSQL, Hibernate (JPA), RESTful API, Maven
  • DevOps & Deployment: Docker, H2 Database for development and testing

Encryption and Decryption Tool - Webpage

Description: Final project for the Oracle Next Education Front-End program with Alura. This project is a simple and intuitive encryption tool that allows users to either encrypt or decrypt a text using a Caesar Cipher method. It features a user-friendly interface where users can input text, apply encryption or decryption, and then easily copy the result.

The Caesar Cipher encryption shifts the letters in the text by three positions, while decryption shifts them back to their original form. The tool only accepts lowercase letters without accents to maintain consistency and prevent errors during the encryption/decryption process.

Skills Implemented:

  • HTML5: Structuring content with semantic HTML elements, creating a responsive user interface.

  • CSS3: Styling the interface with Flexbox for layout adjustments, applying custom fonts and colors using variables, designing responsive layouts that adapt to different screen sizes.

  • JavaScript: Handling user input and applying Caesar Cipher encryption and decryption logic, manipulating the DOM dynamically to display results and control visibility of elements, implementing localStorage to remember user language preferences, using event handling for buttons.

  • Responsive Design: Ensuring the interface is adaptable across devices using media queries.

  • Language Toggle Feature: Allowing users to switch between Portuguese and English interfaces with stored preferences.

  • Caesar Cipher Algorithm: Applying a shift-based encryption method to alter text, ensuring secure data transformation.
